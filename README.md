# Refactoring-maven-plugin

Repository with a few maven plugin about Refactoring and Code Style

# Maven Commands

To use the demo
mvn clean verify

To install the bugchecker-plugin in the local repository
mvn clean install

To use Checkstyle
mvn site

To use pmd
mvn site
